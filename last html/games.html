Как выбирают игры для киберспорта?

Не каждая игра становится киберспортивной дисциплиной. Для этого игра должна обладать:

  1. Сбалансированным геймплеем
  2. Высокой зрелищностью для зрителей
  3. Возможностью проведения честных соревнований
  4. Поддержкой разработчика
  5. Активным комьюнити игроков
